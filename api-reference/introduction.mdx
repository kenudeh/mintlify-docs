This is the reference documentation for the Dolphin Flashcard API. The enpoints are grouped into Card management, Goals, and Statistics.

Followthese steps to get started:


It allows you to create user accounts, create and manage flashcards, set learning goals, and track performance with your heatmap and streak. Data is available in JSON format only. <br></br>**Note**: Request body parameters are required, but the query parameters are optional. All parameters are case-sensitive.
